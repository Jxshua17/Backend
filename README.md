# E-Commerce Website's Backend

This is the backend for an [E-commerce website](https://github.com/jxshua17/frontend). 
Where users can 
1. **log in**, 
2. **read** items that 
are available on the website as well as, 
3. **create**
items themselves, 
4. **update** items, 
5. **delete** items, 
6. **add** items to their personal carts. 

New users can also sign up.
